## Table Columns Explained

| Column | Description | Required? |
|--------|-------------|-----------|
| **Storage / Color** | Edit both Storage and Color directly in this column | Yes (at least one attribute) |
| **Price** | Regular price for this variation | Yes (marked with *) |
| **Special Price** | Discount/offer price (optional) | No |
| **Stock** | Available quantity for this variation | Yes (marked with *) |
| **Image** | Upload one image for this variation (optional) | No |
| **Delete** | Remove this variation | - |

---

## Tips

- **Storage field:** Enter values like "128GB", "256GB", "512GB"
- **Color field:** You can type the color name (e.g., "Blue", "Midnight") OR use the color picker to select and it will auto-convert to color names
- **Price:** Set different prices for different storage sizes (e.g., 256GB = 145000, 128GB = 140000)
- **Stock:** Each variation has its own stock. Total stock is automatically calculated.
- **Image:** Each variation can have one image. Upload the image showing that specific color/variant.
